```bash

cd Mindly -react
```

```bash
npm install
```

### 3. Jalankan Development Server

```bash
npm run dev
```

Aplikasi akan berjalan di `http://localhost:5173`

### 4. Build untuk Production

```bash
npm run build
```

Hasil build akan tersimpan di folder `dist/`

### 5. Preview Production Build

```bash
npm run preview
```
